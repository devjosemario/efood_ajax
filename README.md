# eFood (AJAX + Modal + Carrinho/Checkout)

Fluxo igual às telas:
1) Carrinho (drawer lateral)
2) Entrega (form)
3) Pagamento (form)
4) Confirmação (pedido realizado)

## Como rodar
1) npm install
2) npm run dev

## Como testar o fluxo
- Clique em "Comprar o produto" -> abre modal
- "Adicionar ao carrinho" -> abre carrinho (tela 1)
- "Continuar com a entrega" -> tela 2
- "Continuar com o pagamento" -> tela 3
- "Finalizar pagamento" -> tela 4
- Botões de voltar seguem o pedido do enunciado
